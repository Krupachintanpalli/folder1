CSS Structure and Rules

  • Inheritance

  • Virtually all selectors which are nested within
    selectors will inherit the property values assigned to
    the outer selector unless otherwise modified.
  • For example, a color defined for the BODY will also
    be applied to text in a paragraph.

  • Comments

  • Comments are denoted within style sheets with the
    same conventions that are used in C programming.
    A sample CSS1 comment would be in the format:

    • /* COMMENTS CANNOT BE NESTED */
